\section{测试、运行情况}

\subsection{测试情况}

本程序的每一个实验模块由组员完成后，组长会进行代码审核与测试，
如果发现问题则要求继续修改，直到所有问题被解决后该实验模块才会发布。
我们还建立了用户 QQ 群，并即时反馈用户提出的任何问题。

另一方面，各种 API 的编写与模块化编程也让我们的程序在编写过程中更不容易出错，
同时规范、统一的码风也让调试变得轻松。
